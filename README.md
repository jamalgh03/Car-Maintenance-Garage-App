# Car Maintenance Garage App

## Project Overview
Car Maintenance Garage App is a mobile application designed to manage car maintenance service bookings in a garage.  
The application allows users to book maintenance appointments, track their vehicle maintenance history, and choose the type of maintenance service they need.  
It also provides an admin interface to manage the garage and view general statistics.

---

## User Features
- Book a car maintenance appointment on a preferred date  
- Select the type of maintenance service  
- View and track maintenance history  
- Rate the application after using the service  

---

## Admin Features
- Admin login system  
- View general statistics about the garage  
- Monitor maintenance bookings and activities  

---

## Technologies Used
- Android Studio  
- Java  
- XML  
- PHP  
- MySQL  
- Volley Library  

---

## Project Purpose
The purpose of this project is to simplify the process of booking and managing car maintenance services while providing both users and administrators with an easy-to-use and efficient system.
